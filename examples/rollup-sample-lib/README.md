# rollup-sample-lib

This example exercises the Zephyr Rollup plugin in a small TypeScript library.

## Building

From the repository root, run:

```bash
pnpm --filter rollup-sample-lib build
```

## Running unit tests

Run the unit tests with [Rstest](https://rstest.rs/):

```bash
pnpm --filter rollup-sample-lib test
```
