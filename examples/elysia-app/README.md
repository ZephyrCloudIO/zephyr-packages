# elysia-app on Zephyr

# Setup

Follow these steps to run [Elysia.js](https://elysiajs.com) under [Zephyr Cloud](https://zephyr-cloud.io):

1. Install dependencies
   ```bash
   pnpm i
   ```
2. You're ready to go. Checkout the scripts inside [package.json](./package.json)!
   ```bash
   pnpm build
   ```
